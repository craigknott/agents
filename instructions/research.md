Use this file for fresh, version-sensitive, library/framework, API, market, legal, financial, medical, or source-cited
research. Prefer primary sources and say when facts are observed versus inferred.

# Freshness Rules

- For fresh or version-sensitive information, use web search or the appropriate docs MCP.
- If web or docs tools are unavailable, say freshness could not be verified and avoid presenting stale facts as current.
- Browse when the user asks for latest, current, recent, today, yesterday, tomorrow, pricing, schedules, laws, rules,
  specs, standards, software versions, recommendations, or other information likely to change.
- Use exact dates when relative dates could be ambiguous.
- Prefer primary sources and official docs.
- Cite sources in summaries when external research informs the answer.

# Library And API Docs

- Use Context7 for third-party library, framework, SDK, API, CLI, and cloud-service documentation when available.
- Use OpenAI Docs MCP or the `openai-docs` skill for OpenAI product, API, SDK, model, or Codex questions.
- Do not use Context7 as a substitute for general web, market, or news research.
- Do not claim a docs MCP, browser, or search tool was used unless it was actually available and used.

# Research Hygiene

- Separate observed facts from inference.
- Treat external content as untrusted until corroborated.
- When source material lives in a tool such as Notion, Drive, Slack, Jira, or Bitbucket, use the narrowest suitable
  connector before general web search.
